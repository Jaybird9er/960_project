get started on git

This guide was written to help new coders get started using Git right away. The guide will not get you all the way there, but just far enough along to begin using it on your own. In this way, you can start to learn some of the basics of how to use Git while you learn web or software development. To be clear, this is part one of a two part guide. The second guide will be designed to help users prepare for working on larger projects and in groups. For now, this should be enough for very early beginners.

step 01

get started on GitHub and Git

  1. Sign up for an account on GitHub.
  2. Create a New Repository (project) on Github.

    Add repository name. You can use something like, my_repo to practice, or use a name for a current project. Add a description. Hit Create Repository.


    Ideally, it would be best to check if Git is already installed on your computer. However, since this is a guide for beginners, we can skip that step.


  3. To install Git, go here.

    For Windows, it's best to select the Git Bash option during installation to install a command line app. For Mac, it's best to use the Git for Mac Installer.
